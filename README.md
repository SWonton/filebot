# filebot

Version 1.0.0

Prerequisites: Node.js (and npm), youtube-dl

LINUX ONLY

## Description

General use Discord bot for remote server file management. I run it on a Raspberry Pi Zero.

### Command list

`ls`
`sh` (execute linux command)
`newmusic`
`mkdir`
`rename`
`upload`
`retrieve`
`rm`
